Mapping Purpose

This mapping reads a list of ISIN (International Securities Identification Numbers) from a CSV file, extracts only the ISIN column, creates a list of distinct/unique ISIN values, and outputs:

  1. The distinct ISIN list to a file (Distinct_ISIN_list)
  2. The total count of records to a staging database table (STAGING/DPR_DUMMY_TARGET)

The mapping performs deduplication of ISIN values and provides both the cleaned data and metrics about the processing.
