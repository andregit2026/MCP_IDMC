# Output Directory

This directory contains all generated output files from IDMC MCP Server scripts.

## Current Files

### API Comparison Analysis
- **`API_v2_vs_v3_Comparison.md`** - Comprehensive technical comparison between REST API v2 and Platform API v3
- **`API_Comparison_Summary.txt`** - Quick reference guide with visual comparison tables
- **`mapping_v2_response.json`** - Full REST API v2 response for mapping `m_arausch_DPR_DISTINCT_ISIN3`

### Mapping Analysis
- **`m_arausch_DPR_DISTINCT_ISIN3_analysis.html`** - Visual HTML report of mapping transformations and data flow

## Generated By

All files are generated by scripts in `../scripts/`:
- `compare_api_versions.py` - API comparison tool
- `generate_mapping_report.py` - Mapping analysis report generator

## Usage

All scripts in this project are configured to output their results to this directory automatically.
